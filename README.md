# mai-family — 미래에셋 AI 패밀리

MALife-AI의 **미래에셋 AI 패밀리** — 마법 테마의 이름을 가진 **10개 제품**의 묶음입니다.
지식 관문 **MAGE**를 중심으로, 일손을 돕는 AI 도구와 협업·개발 플랫폼을 하나의 패밀리로 잇습니다.
모두 **폐쇄망 전용**으로, 코드·자료가 회사 밖으로 나가지 않습니다.

## 제품

| 영역 | 제품 |
| --- | --- |
| 지식 관문 | **MAGE** |
| AI 도우미 | **MAGIC**(데스크톱) · **MOJO**(터미널) · **MERLIN**(IDE) |
| 분석·학습·활용 | **MYSTIC**(대시보드 챗봇) · **MARK**(학습 엔진) · **MANA**(활용 아카이브) |
| 협업·개발 | **MESH**(지식·문서, 노션형) · **MAGNET**(대화·협업, 슬랙형) · **MAGMA**(Git 플랫폼) |

## 페이지 · 발표자료

- **`index.html`** — 제품 소개 랜딩(패밀리 지도·제품 카드·공통 강점). 외부 의존성 없는 단일 페이지, 라이트/다크 테마.
- **`decks/`** — 제품별 자기완결형 HTML 발표 덱(폰트·로고·실제 화면 내장, 오프라인·폐쇄망 OK). `← →` 로 넘깁니다.
