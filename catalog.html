패밀리 지도 · Family Map

MAGE 지식 관문을 중심으로 모이는 10개 제품

각 노드를 누르면 해당 제품 카드로 이동합니다. 사내 AI 도구는 MAGE 게이트웨이를 통해 사내 sLLM에 연결되고, 협업·개발 제품(MESH·MAGNET·MAGMA)이 일하는 공간을 채웁니다.

MAGE = 지식 관문(허브) AI 도구 분석·학습·활용 협업·개발 (신규)

제품군 · The Seven

각 제품의 정체와 스택

MAGE허브
MiraeAsset AI Governance Engine

사내 문서·AI·에이전트를 안전하게 다루는 서버 플랫폼. 문서 인제스트→GraphRAG→에이전트 추론을 사내에서 처리하고 IAM·DLP·감사를 강제하는 거버넌스 엔진.

Python 3.11FastAPILangGraphChromaDBNetworkXvLLM·litellm

패밀리의 중심 관문 — 사내 AI 도구가 연결되는 게이트웨이와 도구 표준(MCP)을 제공하고, IAM·DLP·감사를 강제.

MAGIC데스크톱
MiraeAsset AI Generative Intelligence Console

직원용 데스크톱 AI 비서. VDI 친화적 Tauri 셸로 사내 MAGE 서버에 연결해, 문서·업무를 대화로 처리.

RustTauri 2React·Vite

가벼운 데스크톱 셸 + MAGE 연동 브리지로 사내 AI를 바로 손에.

MOJO터미널
MiraeAsset On-prem Job Operator

사내 AI 코딩 CLI. 터미널에서 코드를 읽고 고치고 실행 — MERLIN 브리지와 MAGE 게이트웨이로 사내 sLLM에 연결.

TypeScriptNode ≥22React·Ink

사용자 키가 밖으로 새지 않도록 자격증명을 봉인하고 사내 게이트웨이로만 추론.

MERLIN자체 개발
MiraeAsset Eclipse Reasoning LLM INtegration

Eclipse/PRO-frame IDE 임베디드 AI 코딩 위저드. 레거시 제약(JDK 1.6 · Eclipse 3.7) 위에 RAG+tool use+diff 수락을 이식.

Java · JDK 1.6Eclipse 3.7Lucene 4.10sqlite
규모 ~21.8K LOC

패밀리 유일 순수 자체 개발. MERLIN→MOJO(서브프로세스)→MAGE 경로로 연동.

MYSTIC분석
MiraeAsset Yield-aware Smart Tableau Insight Chatbot

Tableau 대시보드 위 플로팅 챗봇(Chrome 확장) + FastAPI + Tableau MCP 서버. 사내망 오프라인 배포 전제.

PythonFastAPITableau MCPChrome 확장

대시보드를 말로 묻는 인사이트 — Tableau 화면 위에서 바로 질의응답.

MARK학습
MiraeAsset AI Adaptive Retraining Kernel

이름과 달리 LLM 재학습이 아니라 concept-drift 온라인 학습 라이브러리. numpy 단일 의존성의 소형 연구 증류물.

Pythonnumpy onlyMCP stdio

가벼운 concept-drift 온라인 학습 라이브러리 — 변화하는 데이터에 모델을 맞춰가는 커널.

MANA활용
Mirae Agent & Network Archive

Claude Code 스킬팩 — 에이전트·스킬·룰·커맨드를 ~/.claude/에 원라인 설치. 실제 연결점은 패밀리가 아니라 Claude Code.

MarkdownBash·PowerShellPython 유틸
구성 에이전트 141 · 스킬 18

필요한 AI 기능을 원라인으로 설치 — 에이전트·스킬·룰·커맨드를 꺼내 쓰는 아카이브.

MESH신규 · 협업
Mirae Enterprise Sync & Handbook · 사내 노션

지식·문서 허브. BlockNote 기반 자체 구축(외부 SaaS·CDN 미의존), 문서·위키·핸드북을 폐쇄망에서 작성·공유. AI 요약·검색은 MAGE 게이트웨이 연동.

Next.js 15 · React 19BlockNote폐쇄망 전용
저장 로컬 JSON · AI openai-compatible

외부 SaaS·CDN에 기대지 않는 자체 구축 문서 허브 — 폐쇄망에서 작성·검색·공유.

MAGNET신규 · 협업
Mirae Async Group Network & Event Threads · 사내 슬랙

대화·협업 메신저. Rocket.Chat(MIT) 사내 리브랜딩, 채널·스레드·DM. /ai 질의응답·채널 요약은 사내 LLM 게이트웨이 연동.

Rocket.Chat · Meteor브랜드 리스킨폐쇄망 전용
AI /ai ask·요약 · 차단 클라우드 등록/업데이트/푸시

air-gapped 기본값 — 클라우드 등록·업데이트 체크·푸시·통계 전송을 소스/compose에서 전부 off.

MAGMA신규 · 개발
Mirae Automated Git, Merge & Artifacts · 사내 Git 플랫폼

두 축 — 서버(GitLab 벤치마크, Forgejo 기반: Git 호스팅·MR·CI/CD·레지스트리)와 클라이언트(GitKraken 벤치마크, Tauri 기반: 커밋 그래프 GUI). 소스→머지→빌드→아티팩트.

Forgejo/GiteaTauri 데스크톱폐쇄망 전용
서버 docker-compose(air-gapped) · 클라이언트 .exe

코드·파이프라인·아티팩트 모두 사내에서만 — 외부 egress·업데이트·텔레메트리 off. 스캐폴딩 server/·client/ 커밋됨.

공통 설계 · Shared DNA

패밀리 전반을 관통하는 4가지 철학

01

폐쇄망 제일주의

전 제품이 외부 API를 쓰지 않고 사내 게이트웨이로만 추론을 라우팅. 코드·자료가 밖으로 나가지 않는 게 공통 셀링포인트.

02

하나의 관문

MAGE가 사내 AI 도구들이 말을 거는 공통 규격을 제공 — 이질적 클라이언트를 하나로 묶는 접착제.

03

MCP 통일

도구·확장 표준을 MCP로 통일해 제품 간 확장을 일관되게 연결.

04

거버넌스 내장

IAM·DLP·감사·스킬 검수를 관문에 내장 — 안전하게 다루는 것이 기본값.

공통 강점 · By Design

패밀리가 공유하는 설계 강점

폐쇄망 전용외부 egress·업데이트·텔레메트리 차단이 기본값 — 코드·자료가 사내에 머뭅니다.
하나의 관문사내 AI 도구가 MAGE 게이트웨이로 모여 일관된 규격으로 사내 sLLM에 연결.
거버넌스 내장IAM·DLP·감사가 관문에 내장 — 접근 통제와 활동 기록이 기본.
자격증명 봉인사용자 키가 사내 게이트웨이로 새지 않도록 설계.
MCP 표준도구·확장을 MCP로 통일해 제품 간 연결을 일관되게.
